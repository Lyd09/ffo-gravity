# **App Name**: Redshift Design

## Core Features:

- Logo Display: Display the logo with a red diamond shape.
- Navigation: Display navigation links: About, Gallery and Contact
- Main Title: Display title 'DESIGN' and description 'Our hobby is a modern and convenient design, the key to successful communication with the client.'
- Main Image: Enable the display of the main image with statue in front of red circle
- Carousel: Navigation button to navigate to different images/sections.

## Style Guidelines:

- Primary color: Vibrant red (#FF4136) to reflect a dynamic, bold design aesthetic, referencing the design's dominant visual element.
- Background color: Dark gray (#1A1A1A), near-black but slightly lighter, allowing red elements to pop without being as stark as true black. HSL(0, 0%, 10%)
- Accent color: Off-white (#F0F0F0) to provide contrast on the dark background, ensuring readability. Chosen as a lighter, less harsh alternative to pure white.
- Font: 'Montserrat' sans-serif for headings and body, providing a modern and readable aesthetic. Note: currently only Google Fonts are supported.
- A split-screen layout, presenting concise textual information on the left, complemented by captivating image content on the right.
- Minimal and modern icons with a bias towards geometric shapes.
- Subtle transition effects for image changes
